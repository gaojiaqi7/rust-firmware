### Set RUST_FIRMWARE_FSP_FD_FILE

RUST_FIRMWARE_FSP_FD_FILE need be set.

example

```
export RUST_FIRMWARE_FSP_FD_FILE=$PATH_TO_QEMU_FSP/QEMU_FSP_RELEASE.fd
export EDK2_PATH=$PATH_TO_EDK2
```

example windows cmd

```
set EDK2_PATH=C:\Users\xlu2x\work\edk2
set RUST_FIRMWARE_FSP_FD_FILE=QEMU_FSP_RELEASE.fd
```
